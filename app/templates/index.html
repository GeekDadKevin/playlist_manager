What sync setup does

This tab does not start anything by itself. It simply explains what the app needs before the live sync step can match songs, download missing tracks directly from Deezer, and then update the Navidrome playlist file for you.

  • Set DEEZER_ARL to a valid Deezer session cookie.
  • Point DEEZER_DOWNLOAD_DIR at the folder where new files should be saved.
  • Optionally tune DEEZER_QUALITY and DEEZER_MATCH_THRESHOLD.
  • Once configured, importing a playlist jumps straight to Sync status, triggers downloads, and refreshes the Navidrome playlist file automatically.
